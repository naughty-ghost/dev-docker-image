# リリース・バージョニング規約

本プロジェクトは[セマンティックバージョニング](https://semver.org/lang/ja/)に準拠する。

## タグとリリース名の形式

- **Gitタグ:** `v*.*.*` 形式で付ける（例: `v0.6.14`, `v1.0.0`）
- **GitHub リリース名:** `*.*.*` 形式で付ける（先頭 `v` を付けない。例: `0.6.14`, `1.0.0`）
- タグとリリース名のバージョン部分は一致させる

## バージョン番号の上げ方（0.x 系の慣例）

- **PATCH**（`0.6.13` → `0.6.14`）: 後方互換のバグ修正・内部改善のみ
- **MINOR**（`0.6.x` → `0.7.0`）: 後方互換を壊す変更、または機能追加
- 1.0.0 到達後は通常の SemVer（MAJOR/MINOR/PATCH）に切替
